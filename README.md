🦷 Sistema de Gestión Odontológica

📖 Descripción

Este proyecto consiste en el desarrollo de un Sistema de Gestión Odontológica para el Centro Odontológico Dra. Narabia, diseñado para optimizar la administración de pacientes, citas, tratamientos, facturación y reportes dentro del consultorio.

La aplicación busca centralizar toda la información en una única plataforma, mejorando la organización de los datos, la programación de citas y el acceso al historial clínico de los pacientes. Además, incorpora herramientas de facturación y generación de reportes que facilitan el trabajo administrativo y contribuyen a ofrecer un mejor servicio.

🎯 Objetivo General

Desarrollar un sistema de gestión odontológica que permita mejorar la organización, control y acceso a la información de pacientes, citas y tratamientos en el Centro Odontológico Dra. Narabia.

🎯 Objetivos Específicos
Registrar pacientes en una base de datos organizada.
Gestionar citas de manera automática y eficiente.
Permitir la búsqueda rápida de pacientes.
Almacenar un historial clínico detallado.
Integrar un sistema de facturación.
Generar reportes de pacientes y tratamientos.
Mejorar la organización y acceso a la información.
Reducir errores en la programación de citas.
🛠 Tecnologías Utilizadas
Lenguaje: C#
Framework: .NET
Base de Datos: SQL Server
IDE: Visual Studio Community
Arquitectura: Arquitectura por Capas

🏗 Arquitectura por Capas
🔹 Capa de Presentación (UI)

Contiene los formularios e interfaces gráficas que permiten la interacción de los usuarios con el sistema.

🔹 Capa de Negocio (BLL)

Gestiona las reglas de negocio, validaciones y procesos internos de la aplicación.

🔹 Capa de Datos (DAL)

Administra la conexión con SQL Server y las operaciones de almacenamiento y consulta de datos.

⚙ Funcionalidades del Sistema

👥 Gestión de Pacientes
Registro de pacientes.
Edición y actualización de información.
Búsqueda rápida de pacientes.
Consulta de historial clínico.

📅 Gestión de Citas
Registro de citas.
Reprogramación de citas.
Cancelación de citas.
Validación de disponibilidad de horarios.
Visualización de horarios disponibles.

🩺 Historial Clínico
Registro de diagnósticos.
Control de tratamientos realizados.
Consulta de antecedentes clínicos.

🦷 Gestión de Tratamientos
Registro de tratamientos odontológicos.
Consulta de costos y detalles de procedimientos.

💳 Facturación y Pagos
Generación de facturas.
Registro de pagos.
Control de estado de facturación.

📊 Reportes
Reportes de pacientes.
Reportes de tratamientos.
Consulta de información administrativa.

🔐 Seguridad
Inicio de sesión.
Control de acceso mediante usuarios y roles.
Protección de información sensible.
